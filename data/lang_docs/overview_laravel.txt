What is Laravel?

Laravel is a full-featured, open-source web application framework written in PHP, first released in 2011 and designed around the principle of convention over configuration — meaning it makes strong, opinionated default choices about how a project should be structured so that developers spend less time on boilerplate setup. Laravel ships with an expressive syntax and includes, out of the box, a built-in ORM called Eloquent (for interacting with a database using PHP objects instead of raw SQL), a templating engine called Blade, a command-line tool called Artisan (for generating boilerplate code, running migrations, and other maintenance tasks), a routing system, a middleware pipeline, built-in authentication and authorization scaffolding, a queue system for background jobs, and an event system for decoupling application logic. This "batteries-included" philosophy is what distinguishes Laravel from a minimal framework like Gin: rather than assembling a stack from separate libraries, most Laravel applications rely primarily on tools that already ship as part of the framework itself. Laravel is one of the most popular frameworks for building web applications and APIs in PHP.
