What is Java?

Java is a general-purpose, object-oriented programming language originally released by Sun Microsystems in 1995 and now maintained by Oracle. Java code compiles to an intermediate form called bytecode, which runs on the Java Virtual Machine (JVM) rather than being compiled directly to native machine code — this is the basis of Java's "write once, run anywhere" philosophy, since any platform with a compatible JVM can run the same compiled bytecode. Java is statically typed and strictly object-oriented: nearly everything in a Java program is defined inside a class, and Java supports classical inheritance, interfaces, generics, and a large standard collections framework. Java is widely used for large-scale enterprise applications, backend web services, and Android mobile development (Android apps are traditionally written in Java or Kotlin, both of which run on JVM-like runtimes). Java's long history and huge ecosystem mean it has mature tooling, extensive libraries, and frameworks like Spring and Spring Boot built specifically for enterprise and web development.
